# Elma Oliveira Makeup 💄✨

Site institucional desenvolvido para a marca **Elma Oliveira Makeup**, maquiadora profissional especialista em noivas, com foco em sofisticação, experiência do usuário e posicionamento premium.

Este projeto foi criado do zero utilizando **HTML5 e CSS3**, aplicando conceitos de identidade visual, tipografia personalizada e layout responsivo.

---

## 🔥 Objetivo do Projeto
Criar um site elegante e funcional para:
- Apresentar a profissional e seus serviços
- Destacar experiências exclusivas para noivas
- Disponibilizar materiais em PDF (guias e formações)
- Fortalecer a presença digital da marca

---

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  
- Google Fonts  
- Fontes personalizadas (@font-face)  
- Layout responsivo (mobile-first)  

---

## 🎨 Destaques do Projeto
- Identidade visual premium  
- Tipografia personalizada para a marca  
- Seções bem definidas e organizadas  
- Cards de experiências com imagens  
- Botões para download de PDFs  
- Design responsivo para desktop e mobile  

---

# Elma Oliveira Makeup 💄✨

Institutional website developed for the brand **Elma Oliveira Makeup**, a professional makeup artist specialized in brides, focused on sophistication, user experience, and premium brand positioning.

This project was built from scratch using **HTML5 and CSS3**, applying concepts of visual identity, custom typography, and responsive layout.

---

## 🔥 Project Goal
To create an elegant and functional website to:
- Present the professional and her services
- Highlight exclusive bridal experiences
- Provide downloadable PDF materials (guides and professional training)
- Strengthen the brand’s digital presence

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Google Fonts  
- Custom Fonts (@font-face)  
- Responsive layout (desktop and mobile)

---

## 🎨 Project Highlights
- Premium visual identity  
- Custom typography for brand consistency  
- Well-structured and organized sections  
- Experience cards with images  
- PDF download buttons  
- Fully responsive design
